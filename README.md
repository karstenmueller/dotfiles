# Personal dotfiles

This is a place to store all your dotfiles and make 127.0.0.1 your $HOME.

**Fork this repository to your personal GitHub space and work from there.**

Just clone dotfiles and install them:

~~~zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/karstenmueller/dotfiles/main/script/install.sh)"
~~~

## Use with GitHub Codespaces

To use these dotfiles with [GitHub Codespaces](https://github.com/features/codespaces) see [Personalizing Codespaces](https://docs.github.com/en/codespaces/customizing-your-codespace/personalizing-codespaces-for-your-account#dotfiles)

## Use with macOS boostrap

To bootstrap a fresh install of macOS and install these dotfiles see [macOS bootstrap](https://github.com/karstenmueller/macos-bootstrap)
